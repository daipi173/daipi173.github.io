工作篇——ubuntu相关的安装（三）

“ 最近工作已经一个月了,也遇到了相关的一些问题(ps:两次重装系统的经历让我成为了一个无情的安装软件的工具人),并且备受打击的是之前写的几篇文章由于一些原因无法找回,在这里我就总结一些常见的错误,等我想到并且还遇到一些谷歌搜索不到的问题时,我会再开一篇文章来总结”

“当然如果你在ubuntu系统中下载软件时,遇到不能解决的问题,也欢迎在下面评论,如果我看到能够解决的话也会及时的做出解答。”

1. Ubuntu系统安装软件之跳板机

如何配置跳板机？（ps：对于不同公司有不同的配置操作，下面步骤仅作参考）

（1）首先去公司提供的虚拟机申请地址里面去申请；

（2）当公司审批过之后，会发相关的企业邮件（在这里以outlook邮件为例）给你，找到企业邮件，在里面你会找到相关的id_rsa（即私钥，这个私钥很重要，需要自己放好不要给别人）文件和相关的密码；

（3）将第二步骤中的id_rsa文件下载到你想要的路径下，然后找到该路径的位置，右键在终端中打开，接着执行指令sudo chmod 400 id_rsa将它的权限改为只读权限（如果你没有修改文件的权限直接进入第四步的话，终端会给你报warning，并且也无法登录到ssh）；

（4）在第三步将权限修改完之后，我们接着输入指令登录ssh，ssh -i id_rsa 自己名字@ip地址(注意id_rsa看你具体放在什么位置，根据你放置的位置来写)；

（5）接着会让你输入相关的密码，这个密码就是你第一步中企业邮件中的密码，将密码复制再粘贴到终端中，接着就登录成功了；

（6）接下来你就登录跳板机，跳到相关的服务器，输入指令ssh 虚拟机的主机名，然后让你需要密码，密码也是第一步中企业邮件中的密码；

如何配置免密登录跳板机？

（1）当你输入指令登录ssh，ssh -i id_rsa 自己名字@ip地址，在进入到ssh之后，首先创建.bashrc文件，执行指令touch .bashrc；

（2）编辑该文件，执行指令vim .bashrc，然后将eval /usr/local/bin/ssh-attach这句话复制到.bashrc文件的最底部；

（3）再执行指令source .bashrc让该文件立即生效（source的作用：通常用于重新执行刚修改的初始化文件，使之立即生效，而不必注销并重新登录）；

（4）当第三步生效之后，接着输入指令ssh 主机名登录服务器，这是如果没有让你密码登录，而是直接进入的话，证明你配置成功了。

2. Ubuntu系统安装完使用搜狗时遇到的坑

在公司使用该linux系统也才满一个多月,但是在开发中也会遇到比较头疼的事情,就比如和我一起开发的小伙伴的搜狗在下载不到多久就罢工了,重新安装之后也无法使用,而我遇到的问题和他相近,在开发中忽然就从搜狗拼音变为搜狗五笔(ps:对于我这种不会打五笔来说就是煎熬)。后面我尝试点击桌面右上角搜狗图标中重新启动,之后就变为乱码或者不显示了。
对于这种情况，我去谷歌搜索了很久也没有找到相关的解答，接下来我就讲解一下自己是如何解决这样的问题的。
首先声明一点：我在下载搜狗软件时是通过deb包的方式下载的，下面对于卸载软件时是通过这种方式来卸载的，如果你是用其他方式来下载的，那么你可以去搜索一下如何卸载干净该软件。
解决步骤：
完全卸载搜狗输入法：

一般执行指令：sudo dpkg -r sogoupinyin，如果这个时候还没有卸载干净的话接着执行下一步；

卸载fcitx：sudo apt-get purge fcitx；

彻底卸载fcitx及相关配置：sudo apt-get autoremove；

执行完上面三步之后搜狗就被完全卸载了，这个时候你点击桌面右下角的显示应用程序，接着去寻找fcitx会发现已经没有了，证明已经卸载完毕了。
重新安装搜狗输入法：

这个可以参考我写的第一篇文章：工作篇——ubuntu相关的安装（一）

删除搜狗输入法中的搜狗五笔(做绝一点，免得到时候它又冒出来)：

首先打开桌面右上角搜狗图标中的配置，然后我们可以看到会有三个输入法；

接着我们找到搜狗五笔，接着点击那个减号—按钮，这个时候我们看到只留两个输入法，分别是键盘-汉语、搜狗拼音；
